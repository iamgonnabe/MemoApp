## 개선점
폴더 이름을 변경할 때 textfield의 크기를 줄이니깐 text가 보이지 않는 문제점이 발생한다.
room db에 이미지 리스트 타입의 컬럼을 추가하는 과정에서 오류가 발생해 이미지를 불러오지만 추가하는 코드는 작성하지 않았다.
메모 페이지에서 체크박스 기능과 그림 그리기 기능은 구성하지 않았다.
매모 페이지에서 메모를 삭제할 때 오류가 발생한다.
